安装和使用

1 通过 NPM 安装： 
npm install number-counter-animation --save
2 在 HTML 中添加元素： 
<div class="counter" data-counter data-target="100">0</div>
3 在 JavaScript 中初始化： 
// ES6 模块导入
import NumberCounter from 'number-counter-animation';

// 初始化所有带有 data-counter 属性的元素
NumberCounter.init();

// 或者使用自定义选项初始化
NumberCounter.init({
  selector: '.my-counter',
  duration: 2000,
  delay: 0,
  separator: ','
});
